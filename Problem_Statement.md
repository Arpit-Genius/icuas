## Introduction
The problem statement of this mini project is a simplified version of the problem statement of ICUAS 2026.  
You can check the official icuas 2026 repo - 'https://github.com/larics/icuas26_competition'. 

## Problem Statement
Find all the aruco markers in the 'ICUAS26_1' map in minimum possible time. The official problem statement has built in topics to publish aruco IDs and position when the aruco marker is detected by any drone's camera. But you are required to use the camera feed of the drones to find the ID and position of detected aruco marker.
